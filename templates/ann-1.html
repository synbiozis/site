Why are we using neural networks ?

Neural networks appeared to be very good in such fields as prediction. What there is to remember, is that the algorithme is able to predict values using the history of this value. For example, when showing a function for example, it can find a pattern and use it to predict what will happend next.

Main Functions

There are only two functions we will be using, each one is represented by an arrow on my sketch.

Update

This function, quite simple actually, will show the result of an entry. On my example, we have only one input (forget the + for now), and one output. We will compute the entry from the input level, through the hidden layer, to the output.

Backpropagate

Now this function is much harder to understand and code. We start with an output given by the user, as a solution to reach. We start from the output, by calculating the error between what was given and what was expected, we then compute that error through the hidden layer unto the input using weights. The second part consist in adjusting the error, and then modifying the weights in order to have a better result.
